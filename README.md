# Gesture-Controlled-Robot
Uses Rapbberry Pi
This project uses computer vision with Mediapipe and OpenCV to control a robot using hand gestures. It uses a Raspberry Pi to control the motors via GPIO.

## Requirements
- Python 3
- OpenCV
- Mediapipe
- RPi.GPIO
